# AI-Powered Garbage Classification System for Sustainable Waste management

This project presents an AI-based garbage classification system using transfer learning with the Xception architecture. The model is trained on the Kaggle Garbage Classification dataset and deployed using TensorFlow Lite for lightweight inference. A Gradio-based interface enables real-time image upload, classification, and waste disposal suggestions.

## Features
- Xception-based deep learning model
- 12-category waste classification
- TensorFlow Lite deployment
- Gradio web interface
- Sustainable waste management focus

## Tech Stack
- Python
- TensorFlow / TensorFlow Lite
- Gradio
- NumPy, Pillow
